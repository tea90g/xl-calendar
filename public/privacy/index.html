XL Calendar Privacy Policy

Last updated: 2026-05-22

XL Calendar is a personal calendar application. This page explains how XL Calendar handles user data when Google Drive synchronization is used.

1. Data Collected by the Developer

The developer of XL Calendar does not collect, store, sell, or share users’ personal data.

2. Google Drive Access

XL Calendar uses Google Drive only for synchronization of the calendar data file created or selected by the user. The app uses limited Google Drive access for the purpose of saving and loading XL Calendar data.

3. Data Storage

Calendar data is stored locally on the user’s device and, when synchronization is enabled, in the user’s own Google Drive. The developer cannot access the user’s Google Drive or calendar data.

4. Data Sharing

XL Calendar does not share user data with third parties.

5. User Control

Users can stop using Google Drive synchronization at any time by disconnecting the app or deleting the stored XL Calendar data from their Google Drive.

6. Contact

For questions about this Privacy Policy, please contact the developer through X: https://x.com/murmurxl

개인정보처리방침

최종 업데이트: 2026-05-22

XL Calendar는 개인용 캘린더 애플리케이션입니다. 이 문서는 Google Drive 동기화 기능 사용 시 사용자 데이터가 어떻게 처리되는지 설명합니다.

1. 개발자가 수집하는 데이터

XL Calendar의 개발자는 사용자의 개인정보를 수집, 저장, 판매, 공유하지 않습니다.

2. Google Drive 접근

XL Calendar는 사용자가 생성하거나 선택한 캘린더 데이터 파일을 저장하고 불러오기 위해서만 Google Drive를 사용합니다.

3. 데이터 저장 위치

캘린더 데이터는 사용자의 기기에 로컬 저장되며, 동기화를 활성화한 경우 사용자의 Google Drive에 저장됩니다. 개발자는 사용자의 Google Drive 또는 캘린더 데이터에 접근할 수 없습니다.

4. 제3자 제공

XL Calendar는 사용자 데이터를 제3자에게 제공하지 않습니다.

5. 사용자 제어

사용자는 언제든지 Google Drive 동기화를 중단하거나 Google Drive에 저장된 XL Calendar 데이터를 삭제할 수 있습니다.

6. 문의

개인정보처리방침 관련 문의는 제작자 X로 연락해 주세요: https://x.com/murmurxl
